## Projects

### Agents & AI

- **[triage-agent](https://github.com/TheGreatAxios/triage-agent)** — Cloudflare Workers agent for incident response. Telegram ingestion, Slack escalation, Linear management.
- **[Prawl](https://github.com/thegreataxios/prawl)** — Browser automation CLI for AI agents. ~63MB self-contained binary with MCP support. Bun.WebView.
- **[ai-sdk-privacy-filter-middleware](https://github.com/TheGreatAxios/ai-sdk-privacy-filter-middleware)** — Vercel AI SDK middleware that redacts PII locally using Transformers.js. No API calls.
- **[privacy-filter-python](https://github.com/TheGreatAxios/privacy-filter-python)** — Local-first PII detection and reversible redaction for NanoBot and OpenAI Agents SDK.
- **[privacy-python-server](https://github.com/TheGreatAxios/privacy-python-server)** — Privacy-first AI server with offline PII redaction.
- **[video-transcription-modal](https://github.com/thegreataxios/video-transcription-modal)** — CPU-optimized transcription on Modal using faster-whisper with INT8 quantization.
- **[agent-skills](https://github.com/thegreataxios/agent-skills)** — Reusable agent skills, workflows, and patterns for AI engineering.
- **[nanobot-railway-template](https://github.com/thegreataxios/nanobot-railway-template)** — One-click deploy template for nanobots on Railway. 5+ platforms, 15+ LLM providers.
- **[llms-txt-generator](https://github.com/TheGreatAxios/llms-txt-generator)** — Utility for generating llms.txt / llms-full.txt from documentation sources.

### Payments & Blockchain

- **[Armory](https://github.com/thegreataxios/armory)** — Opinionated TypeScript x402 SDK. [armory.sh](https://armory.sh)
- **[x402](https://github.com/coinbase/x402)** — Contributor to the open standard for internet-native payments. [x402.org](https://x402.org)
- **[eip3009-forwarder](https://github.com/TheGreatAxios/eip3009-forwarder)** — Production-ready EIP-3009 forwarder contracts for x402 on EVM chains.
- **[x402-examples](https://github.com/TheGreatAxios/x402-examples)** — Comprehensive examples and tests for the x402 payment standard.
- **[confidential-poker](https://github.com/thegreataxios/confidential-poker)** — AI Texas Hold'em with 6 agent personalities. Foundry/Hono/Next.js.
- **[gasless-rs](https://github.com/Dirt-Road-Development/gasless-rs)** — Rust implementation of gasless transaction infrastructure.
- **[skale-cli](https://github.com/TheGreatAxios/skale-cli)** — CLI tools for SKALE network interactions and deployment.
- **[skl-supply-ui](https://github.com/TheGreatAxios/skl-supply-ui)** — Real-time dashboard for SKL token economics and supply metrics.
